oEmbed Support for edocr documents

  • - Request type : HTTP GET
  • - Authorization : none
  • - URL : http://www.edocr.com/api/oembed
  • - Default response format : json

Quick example call

- http://www.edocr.com/api/oembed/?format=json&url=http://www.edocr.com/doc/1465/6-steps-acquire-new-customers-edocr-com

Query Parameters

  • - url (Required) : The URL to retrieve embedding information for
  • - format (optional) : xml, json

Response parameters

  • - version : The oEmbed version number.
  • - type : Type of oEmbed response.
  • - title : Title of the embed document.
  • - author_name : Author of the embed document.
  • - author_url : edocr profile page of the author of embed document.
  • - provider_name : Provider of the embed content, edocr.
  • - provider_url : URL of provider.
  • - html : This is the html embed code, which contains links to embed document.
  • - width : Width of the embed document.
  • - height : Height of the embed document.

Error Codes

  • - 404 Not Found : If URL is missing or doesn't point to an embeddable resource.
  • - 501 Not Implemented : If the format provided is not supported. Should be one of xml, json

Example

XML Request

- http://www.edocr.com/api/oembed/?format=xml&url=http://www.edocr.com/doc/1465/6-steps-acquire-new-customers-edocr-com

JSON Request

- http://www.edocr.com/api/oembed/?format=json&url=http://www.edocr.com/doc/1465/6-steps-acquire-new-customers-edocr-com
